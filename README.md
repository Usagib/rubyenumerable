# Ruby Enumerable methods

### This scripts rebuilds ruby Enumerable methods:

- #each as #my_each
- #each_with_index as #my_each_with_index
- #select as #my_select
- #all? as #my_all?
- #none? as #my_none?
- #count as #my_count
- #map as #my_map
- #inject as #my_inject
